Qualcomm Snapdragon Multiverse Hackathon Architecture Specification

Q
Arduino UNO Q
Deterministic Sensing Node
Acquisition Thread 100ms Loop
STM32 core runs Zephyr OS. Read cell voltages (A0-A3), current (A4), temp (A5), gas (A6), and vibration (MPU6050 via I2C).
DSP Noise Filter
Executes local 5-tap median and moving average filters on the signals to strip high-frequency EV traction motor harmonics.
Bridge Write
STM32 writes output sensor floats to parameters inside shared memory RPC tables via `Bridge.put("voltages", csv_string)`.
Hardware IPC Bus →
QRB2210 Gateway
MPU Debian Linux daemon checks registers via `bridge.get()`. Packages metrics into formal JSON telemetry templates.
Store-Forward & BLE
Saves variables to local eMMC storage when offline. Exposes a Bluetooth 5.1 GATT service node fallback.
💻
Snapdragon X PC
Hexagon NPU Edge Host
Mosquitto MQTT
Ingests packet inputs from the QRB2210 gateway and vehicles on the topic `ev/sensor/telemetry` over local router link.
Ingress Vectorizer backend.py
Parses JSON payload elements. Serializes attributes into a float32 array vector of shape `[1, 8]` for the inference engine.
Trust & SOH Evaluation HTP NPU
Runs ONNX Runtime (QNN EP) on the Hexagon HTP NPU to calculate Trust Score % and State of Health % inside 1.2ms.
WebSocket Server
Broadcasts raw variables, trust assessment outcomes, and calculations to client screens via WebSockets.
Offline Llama-3
Local assistant (Ollama) reads rolling status history logs to provide natural language battery health diagnoses.
📱
Companion App
Driver Mobile Receiver
WebSocket Link
Listens continuously to connection metrics on port 8080. Evaluates incoming variables to refresh screen grids.
Diagnostic Screens
Renders circular SoC% dial controls, temperature color map cells, and State of Health (SOH) history plots.
Safety Alerts
Watches anomaly codes. Triggers audio alerts and warning popups if cell trust scores drop below safety thresholds.
Secondary Radio Link →
BLE Diagnostic Mode
Establishes fallback links to the QRB2210 GATT parameters over Bluetooth 5.1 if local Wi-Fi networks fail.
☁️
Cloud AI 100
Fleet Analytics & ML
HTTPS Ingress API
Receives compressed performance logs from vehicle clients every 24 hours. Accepts 10s incident snapshot blocks.
Fleet Database
Aggregates metrics profiles across the vehicle fleet to support battery degradation and thermal profiling analytics.
Fleet Retraining
Iteratively retrains anomaly models on Cloud AI 100 accelerators to improve anomaly detection precision.
AI Hub Compilation
Compiles and quantizes models to QNN format via the `qai-hub` toolchain targeting the Snapdragon X Elite NPU.
OTA Updates
Pushes updated QNN-compatible model weights to vehicle edge nodes over secure networking links.
