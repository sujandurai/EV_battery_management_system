   ┌───────────────────────────────────────────────────────────────────┐

   └───────────────────────────────┬───────────────────────────────────┘
                                   │
                                   │  Raw digital numbers (0–4095)
                                   ▼
   ┌───────────────────────────────────────────────────────────────────┐
   │          THREAD 1 — SENSOR ACQUISITION  (runs every 100ms)       │
   │                                                                  │
   │   1. Read all 7 ADC channels                                     │
   │   2. Read MPU6050 vibration over I2C                             │
   │   3. Convert raw numbers to real units:                          │
   │        ADC 2912 → 3.82 Volts                                    │
   │        ADC 1515 → 34.5 °C                                       │
   │        ADC  932 → 1.25 Volts (fault!)                            │
   └───────────────────────────────┬───────────────────────────────────┘
                                   │
                                   │  Scaled float values
                                   ▼
   ┌───────────────────────────────────────────────────────────────────┐
   │              TINYML / DSP NOISE FILTER                           │
   │                                                                  │
   │   Removes electrical noise from EV motor interference.           │
   │   Uses 5-tap moving median filter.                               │
   │   Prevents single-spike false alarms.                            │
   └───────────────────────────────┬───────────────────────────────────┘
                                   │
                                   │  Clean, filtered values
                                   ▼
   ┌───────────────────────────────────────────────────────────────────┐
   │       THREAD 2 — BRIDGE WRITER  (runs every 100ms)               │
   │                                                                  │
   │   Formats values into text strings:                              │
   │     Bridge.put("voltages",    "3.82,3.81,1.25,3.83")             │
   │     Bridge.put("temperature", "34.2")                            │
   │     Bridge.put("current",     "-12.4")                           │
   │     Bridge.put("vibration",   "0.08")                            │
   │     Bridge.put("gas",         "12")                              │
   │                                                                  │
   │   Writes these strings into shared memory (RPC Bridge).          │
   └───────────────────────────────┬───────────────────────────────────┘
                                   │
                                   │  Shared memory (hardware bus)
                                   ▼
   ┌───────────────────────────────────────────────────────────────────┐
   │       THREAD 3 — LED MATRIX DISPLAY  (runs every 500ms)          │
   │                                                                  │
   │   Reads Bridge.get("trust_status") from shared memory.           │
   │                                                                  │
   │   If value = "OK"    →  Show battery charging animation ████     │
   │   If value = "FAULT" →  Flash warning symbol  !!  on LEDs       │
   └───────────────────────────────────────────────────────────────────┘
